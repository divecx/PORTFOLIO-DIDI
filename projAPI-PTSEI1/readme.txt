Buatlah 3 buah endpoint REST API dengan Bahasa pemrograman yang dikuasai dengan ketentuan:
a. POST /login
  - Parameter request: username dan password
  - Parameter output: username, name dan email
b. GET /user
  - Parameter request: none
  -Parameter output: userid, username, name dan email dalam bentuk list/array
c. POST /user
  - Parameter request: username, password, name, email
  - Parameter output: userid
